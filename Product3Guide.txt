To retrive saved data from Media Players, Kindly reboot your system.
Compatibility issue of media player with the system, Kindly install application with newer version.
Media players sound can also be manage by clicking Manage tab in the Media Player and clicking adjust sound button 
Media player Files are by default stored in a temporary  location, for finding it go to document folder which is by default set there. 
for noise spikes in media player, kindly volume down the sound.
If media player is giving processor compatibility  error, kindly download correct version of media player.
